# Python LLM Project

This project allows you to enter a prompt, calls an open-source LLM (gpt-oss-20b) via the GitHub Marketplace API, and outputs the project files generated by the LLM.

## Usage

- Run `main.py` and enter your prompt when prompted.
- The generated files will be saved in the `output/` directory.

## Requirements

- Python 3.8+
- Internet access

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Run the project: `python main.py`

## Note

- The prompt is blank by default for you to type.
- API integration details may require your GitHub token or Marketplace credentials.
